Privacy Policy

Last updated: 14 July 2026

tg2fb is an automated cross-posting tool. It reads new posts from a single Telegram channel that its operator controls and re-publishes that same content to a single Facebook Page that the same operator administers. It exists only to mirror the operator's own public announcements from one platform to another.

What data the app touches

The app processes the content of posts from the operator's own Telegram channel: the post text or image caption, and any attached photo. This content is fetched from the Telegram Bot API, optionally translated, has the operator's logo overlaid on images, and is then published to the operator's Facebook Page. The only state the app stores is the numeric ID of the last processed Telegram update, kept so that posts are not published twice.

No third-party user data is collected

The app does not collect, store, sell, or share any personal data belonging to other people. It does not track visitors, does not read Facebook or Telegram user profiles, and does not build user profiles or advertising audiences. It interacts only with the operator's own channel and Page using access tokens that the operator supplies. No analytics, cookies, or third-party trackers are used.

Data retention

The app keeps only the last-processed update ID needed to avoid duplicate posts. Downloaded images are temporary and are deleted after each post is published. No message archive or personal information is retained.
